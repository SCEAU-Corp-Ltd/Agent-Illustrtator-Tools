# Public Data

This directory holds public reference data used by the repository docs.

## Illustrator Menu Commands

`illustrator-menu-commands.csv` mirrors the public `data/menu_commands.csv`
file from Josh Duncan's AiCommandPalette project:

https://github.com/joshbduncan/AiCommandPalette/blob/main/data/menu_commands.csv

The source project is MIT licensed. See `NOTICE.md` for attribution.

Use this CSV as reference data for Illustrator automation experiments. Verify
commands against the active Illustrator version before running them in a real
document.
